Varios estilos se fusionarán en uno solo

Aquí, el color de fondo de la página se establece con CSS en línea, y también con un CSS interno, y también con un CSS externo.

Intente experimentar eliminando estilos para ver como funcionan las hojas de estilo en cascada (intente eliminar primero el CSS en línea, luego el interno y luego el externo).
